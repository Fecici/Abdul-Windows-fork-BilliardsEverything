# Structured Bug/Optimization Tracker

This directory is the working tracker for the Abdul audit.

Use these files first:

- `issues.csv` - spreadsheet-friendly tracker for filtering by severity, memory impact, Windows impact, and area.
- `issues.jsonl` - machine-readable one-issue-per-line tracker.
- `candidate-fixes/` - proposed fix shapes and code snippets. These are not applied to production source.

Markdown reports in `../` are investigation notes. This tracker is intended to be the durable working surface.
